## Introduction

APPEL – Elevator Service ("APPEL", "we", "our", or "us") provides a mobile application designed for elevator maintenance management and service tracking.

This Privacy Policy explains how we collect, use, and protect information when users access and use the APPEL mobile application.

The application is intended for professional use by authorized personnel only, such as elevator technicians, administrators, and maintenance managers.

---

## Information We Collect

### Account Information

When using the application we may collect:

- Name
- Email address
- User role (Admin, Manager, Technician)
- Authentication data required for login

Passwords are not stored in plain text and are handled using secure authentication mechanisms.

### Elevator and Service Data

The application stores operational data required for elevator maintenance management, including:

- Elevator locations and addresses
- Technical specifications of elevators
- Serial numbers and identification data
- Maintenance and service records
- Repair logs
- SIM card information related to elevator communication systems

### Photos and Uploaded Files

Users may upload photographs related to maintenance activities such as:

- Equipment condition
- Technical issues
- Repair documentation

Uploaded files are stored securely and used solely for service documentation.

Future versions of the application may also allow uploading documents such as service reports.

### Chat Communication

The application includes an internal chat feature allowing team members to exchange text messages related to service operations.

Chat messages are stored as part of operational communication records.

### Location Data

The application may access the device GPS location in order to:

- Assist technicians in locating elevator installations
- Provide navigation or mapping features related to service locations

Location data is used only within the operational context of the application.

### Technical Data

The application may automatically collect limited technical information including:

- Device model
- Operating system version
- Application version

This information is used solely to maintain compatibility and ensure proper application operation.

The application currently does not use third-party analytics or crash tracking services.

---

## How We Use Information

We use collected information exclusively to operate and improve the service, including:

- Managing elevator maintenance records
- Enabling communication between team members
- Authenticating users and managing access permissions
- Synchronizing service data across devices
- Providing technical support
- Ensuring system security and preventing misuse

We do not use personal data for advertising or marketing purposes.

---

## Data Storage and Security

Application data is stored using secure infrastructure including cloud database services.

Security measures include:

- Encrypted data transmission via HTTPS/TLS
- Role-based access control (Admin, Manager, Technician)
- Authentication tokens for secure sessions
- Restricted database access

Local data required for offline functionality may be temporarily stored on the device.

---

## Data Sharing

We do not sell personal data to third parties.

Data may be processed by trusted infrastructure providers strictly for the purpose of operating the application, including:

- Database hosting providers
- Cloud infrastructure services
- Mapping services

Data may also be disclosed if required by applicable law or legal authorities.

## Data Retention

If an account is deleted:

- Operational service records may be retained where necessary for legitimate business documentation

Backups may be stored temporarily for disaster recovery purposes.
